# Project Context: Escuela de Cine

## Vision
A professional online course platform for alternative and experimental cinema. The goal is to provide a premium, immersive learning experience that feels like a "cine editorial" rather than a generic LMS.

## Features

### MVP (Current Scope)
- **Authentication**: Login, Register, Password Recovery (Social Auth structure prepared).
- **Dashboard (Mis Cursos)**:
    - List of enrolled courses.
    - Progress tracking.
    - "Continue Watching" quick access.
- **Catalog**:
    - Browse available courses.
    - Filtering and search.
- **Course Detail**:
    - Syllabus view.
    - Enrollment/Purchase options.
- **Course Player**:
    - Video playback.
    - Lesson navigation.
- **Checkout**:
    - Integration placeholders for Stripe/MercadoPago.
- **Profile**:
    - User settings.

### Future Features
- Community/Social features.
- Personalized recommendations.
- Advanced analytics.
- Dark mode toggle (infrastructure to be prepared now).

## Roadmap
1.  **Skeleton Phase**: Structure, Routing, Base Components.
2.  **Integration Phase**: Connect to Strapi, Real Auth.
3.  **Polish Phase**: Animations, Micro-interactions, Content population.
